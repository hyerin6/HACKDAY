# sns_timeline_2            
인터넷 환경에서 사람들은 다양한 관계를 맺고 새로운 컨텐츠를 서로 주고 받습니다.            
관심있는 사람들과의 소셜네트워크를 구성하여 컨텐츠를 생성/전달하고 컨텐츠를 모아 볼수 있는 타임라인을 구성합니다.
